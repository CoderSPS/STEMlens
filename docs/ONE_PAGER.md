# NeuroSTEM Atlas - One Pager

## The Opportunity
Students struggle with STEM conceptual understanding. Most AI education tools ignore real-world learning artifacts (handwritten notes, diagrams, voice explanations, code) and fail to map reasoning to concepts.

## The Solution: NeuroSTEM Atlas

An AI × STEM education platform that:
1. **Ingests multimodal student work** - handwritten solutions, diagrams, audio, code
2. **Reconstructs reasoning** - AI extracts the thinking process from artifacts
3. **Detects conceptual gaps** - identifies specific areas of misunderstanding
4. **Generates adaptive micro-lessons** - tailored explanations, visualizations, interactive questions
5. **Visualizes mastery on a concept atlas** - dynamic graph showing understanding across STEM topics

## Market

**Primary Users:**
- High school STEM learners (14-18)
- Undergraduate STEM students (18-22)
- Self-learners using online courses

**Pain Points:**
- Don't know which concepts they're weak on
- Generic hints that don't reflect their actual work
- Struggle connecting visual intuition with formal notation
- No unified view of their conceptual mastery

## Why NeuroSTEM Atlas Wins

1. **Multimodal AI** - Not just typed answers; understands real student work
2. **Reasoning reconstruction** - Diagnoses *how* students think, not just *if* they're right
3. **Concept atlas** - Visual representation of understanding across STEM
4. **Adaptive learning** - AI-generated lessons tailored to each student's gaps
5. **High-impact angle** - Targets fundamental problem: concept vs. procedure mismatch

## Core Features

- ✅ Custom auth & onboarding
- ✅ Multimodal submission hub (images, audio, code)
- ✅ AI reasoning reconstruction
- ✅ Adaptive micro-lessons with visualizations
- ✅ Concept atlas with mastery tracking
- ✅ Guided problem-solving sessions with AI hints
- ✅ Advanced: LaTeX export, voice concept check, code-to-concept mapping

## Technology

- **Frontend**: Next.js + TypeScript + Tailwind (dark neural theme)
- **Backend**: Fastify + TypeScript + Prisma + PostgreSQL
- **AI**: FastAPI + Python (multimodal LLM, speech-to-text)
- **Deployment**: Docker + Docker Compose

## Timeline

7-day hackathon build:

## Demo Flow

1. User signs up → selects STEM interests
2. Uploads handwritten solution + audio explanation + code
3. System analyzes and reconstructs reasoning
4. Displays conceptual gaps with confidence scores
5. User views personalized micro-lesson
6. Interactive quiz with feedback
7. Dashboard shows concept atlas with mastery
8. Start guided session for step-by-step practice

## Business Model

- B2C: Student subscriptions ($10-20/month)
- B2B: School/university licensing
- Premium: Advanced features (LaTeX export, unlimited lessons)
